# About
Simple ThreeJS starter template with Shader using Vite
<br/> <br/> <img src="https://github.com/kiritoroo/220816_shader-template/blob/master/demo.jpg" alt="drawing" width="500"/>

# Usage
1. clone this repositry
```
git https://github.com/kiritoroo/220816_shader-template.git
```
2. install package
```
npm i
```

3. start
```
npm run dev
```
